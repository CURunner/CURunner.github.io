Create 3D objects from images on Mac!

Download FlashBuild 3D an app I made to use the Object Capture API and Photogrammetry.

Ever wanted to create 3D objects just from pictures of images? With my new FlashBuild 3D app, you can easily make 3D objects with the new Object Capture API from Apple on Mac. This app is simple and easy to use. After installing simply select a folder full of images taken of an object and before you know it you will have a 3D usdz file!

Download FlashBuild 3D for Mac

Please note that this app requires either an Apple Silicon chip or a discreet GPU.

How to use

Taking Photos

For best results. TAKE AT LEAST 20 PHOTOS (MORE IS BETTER). Try to place the object you are taking photos of on a solid background. Slowly move (or rotate if on turn table) around the object taking photos of it at different angles with about 70% overlap. Try to have good lighting too. When you are finished taking photos. Send the photo's to your mac and put them all in one folder that way you can select them in FlashBuild.

Example of Good Object Capture Photos. (Feel free to test the app with these).

-For your first object. Try setting the quality level to low for a faster processing time. If you are happy with the results, a higher quality setting will take longer but yield a better looking model.
